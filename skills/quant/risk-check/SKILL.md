
---
name: risk-check
description: Risk sanity check for a strategy: exposures, tails, leverage, liquidity.
---

When to use:
- You need a risk review for a strategy or portfolio.

Inputs:
- Strategy description, exposures, constraints.

Output:
- Main risks (market, vol, correlation, liquidity, model/data)
- Tail scenarios + failure modes
- Hidden leverage/convexity checks
- Concentration + crowding checks
- Stress tests + monitoring metrics
- Return a risk register + monitoring spec

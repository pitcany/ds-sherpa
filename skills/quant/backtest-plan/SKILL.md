---

When to use:
- You need a robust backtest plan.

Inputs:
- Strategy description, data sources, constraints.

Output:
- Universe and survivorship handling
- Corporate actions + point-in-time data
- Signal timing (when known vs traded)
- Transaction cost/slippage model
- Constraints (leverage, turnover, sector, liquidity)
- Validation: walk-forward, embargo, sensitivity sweeps
- Reporting: plots/tables + red flags
